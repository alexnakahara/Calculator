# Calculator in JavaScript

### Tricks and tips
- Almost same tricks of windows calculator in your operators.
- Double clicks in the "AC"  turn on sound and turn off.
- The keyboard works:
    - "Esc" = to do clear all;
    - "Backspace" = to do clear entry;
    - "Enter" = to do calculation;
